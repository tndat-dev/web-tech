Content Layout Guide

This project uses a responsive 12-column CSS Grid. Each content item has these settings:

  • Grid column span: the number of columns used on desktop, from 1 to 12.
  • Grid row span: the relative number of rows used by the item, from 1 to 4.
  • Extra CSS: optional inline styles such as background:#eef; or min-height:180px;.

On desktop and tablet screens, each item uses the configured column span. On phones, every item automatically spans 12 columns for readability.
